你喜欢吃甜甜圈嘛

你喜欢吃甜甜圈嘛？

喜欢啊？怎么了？

那你 有没有想过 反正甜甜圈吃几口就没了 为什么还要吃呢？
你会因为甜甜圈被吃完而不吃吗？ 不会对吧？

也许在吃完的时候你会感觉到一丝难过，但是快吃完了，会让你感觉甜甜圈更加美味！

  • 人类存在于这世界上只有几十万年，而天上的星星存在已经几* 十亿年了。
  • 你觉得宇宙真的在意你和我嘛？
  • 当然不会在意，我们只是尘埃罢了。
  • 有时候你会开心，有时候你会难过。
    做你喜欢做的事就好了，奔跑，欢笑，哭泣。轻松的活着，享受生活，享受这个世界的美好，
  • 毛姆在《人性的枷锁》中以主角菲利浦的口吻说：“人生没有意义，人活着也没什么目的。一个人生出来还是没有出生，活着还是死去，都无关宏旨。生命似轻尘，死去亦徒然。”“万事万物犹如过眼烟云，都会逝去，它们留下了什么踪迹呢？世间一切，包括人类本身，就像河中的水滴，它们紧密相联，组成了无名的水流，涌向大海。”他还这样写道，人本来就只不过是一个无足轻重的行星上的短暂生命，因此对于“永恒的头脑”来说，一个人一生的痛苦和奋斗只不过是个笑话而已。”
  • 可是有的时候发现生活就是生活，回到家还会遇到烦人的室友，真的超级影响心情。 我很不能接受晚上十二点之后别人睡了，还在洗澡的、不接受把厕所还有玄关灯开着一晚上不关的人
    、不接受马桶用完边上都是脏兮兮的、不接受为什么聊天还要开免提、不接受为什么有的人会再浴缸里面搓脚- -是我的世界观不正嘛。。还是人们常说的每个人都有每个人的习惯，你不能改变别人！
    其实我觉得不要影响到自己就行了，随便你干什么都可以。
  • 写到这里。我明白了，生活就是为了让你努力，努力赚钱，然后一个人住，努力奋斗，让自己过的舒服。
