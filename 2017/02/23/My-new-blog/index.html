创建hexo博客心得

发表于 2017-02-23 | | 阅读次数

1 初始化博客

first follow those steps,initialization

1> create a new file
$ mkdir hexo-blog

2> enter hexo-blog,input this order
hexo init

you can see this words:
INFO Copying data to ~/Documents/workspace/git/hexo-blog INFO You are almost done! Don't forget to run 'npm install' before you start blogging with Hexo!

hexo-blog will has the following file

scaffolds : source :
themes : _config.yml :

3> install program
$ sudo npm install

hexo-blog will be added a new file:
*node_modules:

2 配置博客

path hexo-blog/_config.yml

1> settings
# Site title:
subtitle: description:
author: language: zh-cn
*timezone:

2> deploy (let your homepage can be visited by others)
(# Deployment)
deploy:
type: git
repository: git@github.com:xxx/xxx.github.io.git
branch: master

3> SSH key
anywherre click right mouse,select Git Bash
$ ssh-keygen -t rsa -C "your_email@youremail.com"

come back Github, enter Settings,select SSH and GPG Keys –> New SSH Key

if you have finished, then input
$ ssh -T git@github.com

if you are first using,you can see continue, input yes, You’ve successfully authenticated,but GitHub does not provide shell access

At last,
$ git config --global user.name "your name" $ git config --global user.email "your_email@youremail.com"
